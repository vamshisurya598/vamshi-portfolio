## Deployment Options
### GitHub Pages
1. Create a repository
2. Upload `index.html` and `styles.css`
3. Enable GitHub Pages from main branch
4. Visit `https://yourusername.github.io/repo-name`

### Netlify
1. Go to netlify.com
2. Drag & drop the folder
3. Website will be live instantly
